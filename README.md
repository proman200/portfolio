

 PROMAN Portfolio Website

A modern, animated personal portfolio website built with HTML, CSS, and JavaScript, designed to showcase skills, projects, and contact information in an interactive and visually engaging way.

 🚀 Overview

This project is a responsive single-page portfolio website featuring smooth animations, interactive UI components, and a futuristic design aesthetic. It is ideal for developers, designers, and creatives who want to present their work professionally online.


 ✨ Features

 ⚡ Modern animated landinghero section
 🧭 Smooth navigation with scroll-based sections
 👤 About section with animated profile avatar and stats counters
 🛠️ Skills section with progress bars and icons
 📁 Projects section with flip-card interactions
 📬 Contact form UI with structured inputs
 🌐 Social media integration section
 🎯 Particle canvas background + cursor effects (via JS)
 📱 Fully responsive layout (mobile-friendly)

 ⚙️ Setup Instructions

1. Clone or download the repository

   ```bash
   git clone repo-url
   ```

2. Open the project folder

   ```bash
   cd portfolio
   ```

3. Run the project

    Simply open `index.html` in your browser
     OR
    Use Live Server (VS Code recommended)

---

 🎨 Customization Guide

 Update name in `index.html`

  ```html
  div class=logoPROMANdiv
  ```

 Replace avatar image

  ```html
  img src=your-image.jpg class=avatar-image
  ```

 Edit projects inside

  ```html
  section id=projects
  ```

 Modify skills progress via

  ```html
  data-progress=85
  ```
